# Wheelchair Assistive Robotic Arm

Assistive robotic manipulation system focused on:
- monocular perception
- AprilTag localization
- visual servoing
- grasp planning
- low-cost assistive manipulation

## Structure

- perception/ → vision pipeline
- planning/ → grasp and motion planning
- control/ → arm/gripper control
- calibration/ → camera calibration tools
- configs/ → YAML configuration files
